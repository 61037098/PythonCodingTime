# 7.12 yum 的使用总结

只下载安装包

```shell
yumdownloader --destdir /home/wangbm/ perl-Sys-Virt.x86_64
```

下载安装包及其依赖包

```shell
yumdownloader --resolve --destdir /home/wangbm/ perl-Sys-Virt.x86_64
```
