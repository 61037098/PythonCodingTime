1、按文件类型搜索 加filetype

::

   流畅的python filetype:pdf

2、过滤关键字 用减号

::

   Python协程 -CSDN

3、必须包含某关键字 用加号

::

   linux常用命令 +centos

4、搜索指定网站 加site

::

   Python冷知识 site:python-online.cn

5、链接中包含字符串 加inurl

::

   Python冷知识 inurl:python-online

6、完全匹配搜索结果 加双绰号包裹
